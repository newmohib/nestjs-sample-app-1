## Getting Started

```bash
# install nestjs Globaly
$ npm i -g @nestjs/cli@latest

# create new project
$ nest new nestjs-api
```


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Create

```bash
# create a new module
$ nest g module users

# create a new module
$ nest g controller users

# create a new module
$ nest g service users

# create resource
$ nest g resource users --dry-run
```
